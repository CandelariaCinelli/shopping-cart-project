# 🛒 Shopping Cart App

Aplicación de carrito de compras desarrollada con React, enfocada en la gestión de estado y la arquitectura de componentes en aplicaciones frontend modernas.

---

## 📌 Descripción

Este proyecto simula el funcionamiento de un carrito de compras de un e-commerce, permitiendo visualizar productos, agregarlos al carrito y gestionar sus cantidades.

Fue desarrollado con el objetivo de afianzar conceptos clave de React como manejo de estado global, hooks personalizados y organización escalable del código.

---

## 🧠 Tecnologías utilizadas

- React
- JavaScript (ES6+)
- Bootstrap
- Vite

---

## ⚙️ Conceptos aplicados

- useState para manejo de estado local
- useReducer para lógica compleja del carrito
- useEffect para efectos secundarios
- Custom Hooks (`useItemsCart`)
- React Router para navegación entre vistas
- Separación de responsabilidades (components, services, reducer, hooks)
- Arquitectura modular y escalable

---

## 🧩 Funcionalidades

- Visualización de catálogo de productos
- Agregar productos al carrito
- Eliminar productos
- Modificar cantidades
- Navegación entre catálogo y carrito
- Cálculo dinámico del total
